

# 🎨 Box Shadow Generator

Uma ferramenta web simples para **gerar sombras em CSS (`box-shadow`) de forma visual e interativa**.

O projeto permite ajustar diferentes parâmetros da sombra e visualizar o resultado em tempo real, facilitando a criação de estilos para interfaces web.

A propriedade `box-shadow` permite adicionar sombras em elementos HTML usando valores como deslocamento, desfoque, espalhamento e cor.

---

## 🚀 Demonstração

Você pode testar a ferramenta diretamente pelo navegador:

👉 https://box-shadow-generator-umber.vercel.app/

---

## ✨ Funcionalidades

- 🎛 Ajuste de **posição horizontal e vertical**
- 🌫 Controle de **blur (desfoque)**
- 📏 Ajuste de **spread (expansão da sombra)**
- 🎨 Seletor de **cor da sombra**
- 🔎 Controle de **opacidade**
- 📦 Opção de **inset shadow**
- 👁 Preview em tempo real
- 📋 Copiar automaticamente o código CSS gerado

## 🎯 Exemplo de saída gerada

```css
box-shadow: 10px 10px 15px 0px rgba(0, 0, 0, 0.3);
```

Parâmetros do `box-shadow`:

| Parâmetro  | Descrição               |
| ---------- | ----------------------- |
| horizontal | deslocamento horizontal |
| vertical   | deslocamento vertical   |
| blur       | nível de desfoque       |
| spread     | expansão da sombra      |
| color      | cor da sombra           |
| inset      | define sombra interna   |
